# NUDA
생리대 성분 분석 및 위험도에 따른 개인 맞춤 제품 추천 Android 앱 서비스

## 📌 주요 기능
- 개인별 AI 맞춤 제품 추천
- 상품 별 성분 분석 및 위험도 제공
- 리뷰 AI 분석
- 성분 위험도 및 주의사항 등 상세 설명
- 전체 랭킹 및 검색
- 찜하기 / 관심 상품 관리
- 마이페이지

## 🛠 기술 스택
| 분야 | 기술 |
|------|------|
| Language | Kotlin |
| IDE | Android Studio |
| Network | Retrofit2 |

- UI/UX 구현 및 서버 API 연동
- Figma를 활용한 와이어프레임 및 디자인 제작
